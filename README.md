# 📊 Marketing Funnel & Conversion Performance Dashboard

## 📌 Project Overview
This project focuses on analyzing user behavior across different stages of a marketing funnel using Power BI. The dashboard helps identify conversion rates, customer drop-off points, and opportunities to improve the lead-to-customer journey.

## 🎯 Objectives
- Analyze user movement through the marketing funnel.
- Measure overall conversion performance.
- Identify major drop-off stages.
- Generate actionable insights to improve conversions.

## 🛠️ Tools Used
- Power BI
- CSV Dataset
- Data Visualization
- Marketing Analytics

## 📂 Dataset Information
The dataset contains:
- **user_id** – Unique user identifier
- **stage** – Funnel stage (Homepage, Product Page, Cart, Checkout, Purchase)
- **conversion** – Conversion status (True/False)

## 📈 Dashboard Features

### KPI Cards
- Total Visitors
- Total Purchases
- Conversion Rate

### Visualizations
- Marketing Funnel Analysis
- Users by Funnel Stage
- Stage-wise Conversion Status
- Conversion Outcome Distribution
- Key Business Insights

## 🔍 Key Insights
- Total Visitors: 10,000
- Total Purchases: 225
- Conversion Rate: 2.25%
- Homepage receives the highest traffic.
- Significant drop-off occurs after the Product Page stage.
- Only a small percentage of users complete the purchase process.
- Improving Cart and Checkout experiences can increase conversions.

## 📊 Results
The dashboard provides a clear view of customer behavior throughout the marketing funnel and helps businesses identify bottlenecks affecting conversion performance.
